## Контекст

LinguaCompanion — voice-first AI платформа для изучения английского русскоязычными IT-разработчиками. Только что обновлён backend (новые промпты агентов, исправлен pipeline, добавлен degraded mode). Frontend обновлён (build args для WebSocket URL). Нужно полное тестирование.

## Что было обновлено с последнего тестирования

1. **Pipeline fix**: variants теперь получает corrected текст вместо сырого RU/EN
2. **Companion Agent**: новые промпты — implicit recasting, sandwich method, scaffolding
3. **WebSocket URL**: исправлен с ws://localhost:8001 на wss://api.lingua.creatman.site
4. **Degraded mode**: если агент падает — pipeline продолжает
5. **Rate limiting**: 15 msg/min, блокировка параллельных запросов
6. **Logging**: все агенты логируют

## Часть 2: Text Mode — основной функционал

### Тест 2.1: Отправка текста
- Переключить на Text mode (кнопка "⌨️ Text")
- **Проверить:** появляется поле ввода текста (прямоугольное со скруглёнными углами) и кнопка отправки
- Набрать: `I work on deployment pipeline yesterday`
- Нажать Enter или кнопку отправки
- **Проверить:** сообщение появляется как user bubble (справа, с синеватым фоном)
- **Проверить:** появляется индикатор "Processing..." или typing
- **Ожидание:** через 3-10 секунд companion отвечает
- **Проверить:** ответ companion появляется слева с аватаром
- **Проверить:** ответ НЕ generic "That's interesting!" — должен быть осмысленный ответ по теме
- **Проверить:** companion может естественно переформулировать (implicit recasting): например использует "worked" вместо "work" в своём ответе
- **Скриншот:** диалог

### Тест 2.2: Analyse (Reconstruction + Variants)
- На отправленном сообщении нажать кнопку "Analyse"
- **Проверить:** появляется ReconstructionBlock:
  - Header "💡 Reconstruction"
  - Строка с "−" (красная) — оригинальный текст в кавычках
  - Строка с "+" (зелёная) — исправленный текст в кавычках
  - Примечание с 💬 — объяснение на РУССКОМ языке
- **Проверить:** появляются VariantCards (5 карточек, горизонтальный скролл):
  - Simple (зелёный badge)
  - Professional (синий badge)
  - Casual (оранжевый badge)
  - Slang (розовый badge)
  - Idiom (фиолетовый badge)
- **Проверить:** каждая карточка содержит текст фразы и подсказку "context" (когда использовать)
- **Проверить:** варианты разные (не одинаковые фразы)
- **Проверить:** варианты НЕ содержат русских слов (pipeline fix)
- **Проверить:** каждая карточка имеет кнопки ▶ (play) и + Save
- **Скриншот:** reconstruction block + variant cards

### Тест 2.3: Code-switching (смешанная речь)
- Отправить: `Мне нужно to deploy the changes to staging`
- **Проверить:** companion понимает смешанный текст и отвечает осмысленно
- Нажать "Analyse"
- **Проверить:** reconstruction показывает полный английский вариант (без русских слов)
- **Проверить:** variants тоже полностью на английском
- **Проверить:** explanation на русском объясняет что было исправлено
- **Скриншот:** code-switching результат

### Тест 2.4: Правильный английский
- Отправить: `I deployed the new version to production yesterday.`
- Нажать "Analyse"
- **Проверить:** reconstruction показывает "Perfect!" или текст без изменений
- **Скриншот:** perfect case

### Тест 2.5: Companion контекст
- Отправить: `What do you think about microservices?`

- Отправить follow-up: `Do you prefer monolith or microservices?`

### Тест 2.6: Не-IT тема
